# Instagram Customizer (Firefox版)

Instagramの不要なボタンやセクションを非表示にすることで、インターフェースをカスタマイズすることができるFirefoxアドオンです。

## 概要

この拡張機能は、Instagramウェブ版の表示を自分好みにシンプルに整えるためのツールです。Manifest V3に対応し、Firefox向けに最適化されています。サイドバーのナビゲーションメニューや、画面上の特定の要素を個別に非表示に設定できます。

## Features

設定画面（ポップアップ）から以下の項目の表示・非表示を切り替えることができます。

### セクション
- **フッター**: 画面下部に表示されるMeta関連のリンクやフッター情報を非表示にします。

### ナビゲーションメニュー

- **ホーム**
- **検索**
- **発見（Explore）**
- **リール**
- **メッセージ**
- **通知**
- **作成**
- **プロフィール**
- **バーガー**
- **その他のMeta製アプリ**

### フローティング要素
- **メッセージアイコン**: 画面右下に表示されるフローティング形式のダイレクトメッセージボタンを非表示にします。

## インストール方法（カスタマイズしたい方向け）

現在、このアドオンを一時的に読み込む手順は以下の通りです：

1. このリポジトリのファイルをダウンロードして解凍します。
2. Firefoxのアドレスバーに `about:debugging#/runtime/this-firefox` を入力します。
3. 「一時的なアドオンを読み込む...」ボタンをクリックします。
4. 解凍したフォルダ内にある `manifest.json` を選択します。

## How to Use

1. Firefoxのツールバーにある「Instagram Customizer」のアイコンをクリックして設定を開きます。
2. 非表示にしたい項目のスイッチをオンに切り替えます。
3. Instagramのページに反映されます。

## Tech Stack
- **Manifest V3**
- **Gecko対応**: `browser_specific_settings` を含むFirefox用構成
- **設定保存**: `chrome.storage.local` を使用してユーザーの好みを保持
- **リアルタイム反映**: `content.js` がストレージの変更を監視し、ページをリロードせずにスタイルを適用します
